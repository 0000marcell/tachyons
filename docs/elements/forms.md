# Forms

Tachyons has some basic form control resets to remove default styles for mobile devices.

## Examples

### Input reset

```.html
<form>
  <label for="inputReset" class="f6 db">input-reset</label>
  <input id="inputReset" type="text" class="input-reset">
</form>
```
